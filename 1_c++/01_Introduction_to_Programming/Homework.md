    🟢 Convert Decimal to Binary:

        ✅ 37   =>  0b100101
        ✅ 92   =>  0b1011100
        ✅ 128  =>  0b10000000
        ✅ 243  =>  0b11110011

    🟢 Convert Binary to Decimal:

        ✅ 0b1011       =>  11
        ✅ 0b111001     =>  57
        ✅ 0b10011011   =>  155
        ✅ 0b10100100   =>  164

    🟢 Convert Decimal to Octal:

        ✅ 28     =>  0o34
        ✅ 47     =>  0o57
        ✅ 928    =>  0o1640
        ✅ 1243   =>  0o2333

    🟢 Convert Octal to Decimal:

        ✅ 0o41     =>  33
        ✅ 0o207    =>  135
        ✅ 0o124    =>  84
        ✅ 0o311    =>  201

    🟢 Convert Decimal to HexaDecimal:

        ✅ 317    =>  0x13D
        ✅ 41     =>  0x29
        ✅ 14     =>  0xE
        ✅ 845    =>  0x34D

    🟢 Convert HexaDecimal to Decimal:
    
        ✅ 0xA11    =>  2577
        ✅ 0x49     =>  73
        ✅ 0xAE2F   =>  44591
